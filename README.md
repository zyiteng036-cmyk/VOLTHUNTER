# VOLTHUNTER

▶ [デモプレイ動画（YouTube）はこちら](https://youtu.be/sGv9UuU36wQ)

## 💻 プロジェクト概要

**ゲーム概要:** 攻撃をジャストでかわして切り返す 電脳世界を駆ける切り返し爽快アクション！
**コンセプト:** 最大のピンチを最高のリターンにしろ！
**開発環境:** Unreal Engine 5 (C++)
**制作期間:** 2025年11月中旬～2026年3月上旬（約4ヶ月）
**チーム人数:** 4人（プログラマー2人、プランナー1人、デザイナー1人）
**担当:** 【プログラム】プレイヤー・カメラ制御全般、UI実装（設定/Clear画面）、Spline移動（床配置・電気移動）、ボス追尾攻撃AI、リスポーン機能
【デザイン・アート】Niagaraエフェクト作成（一部除く）、設定画面UI、床・壁のStaticMesh作成

## 💡 技術的なこだわり・アピールポイント

本作はアクションゲームとしての「直感的な手触り」と「チーム開発における拡張性の高さ」を両立するため、単なるフラグ管理ではなく、**C++によるコンポーネント指向と数学的アプローチ（空間ベクトル演算）**をコアシステムに組み込んでいます。

### 1. 空間ベクトル演算を用いた「ジャスト回避」からの背後ワープ

【課題】 敵の攻撃をギリギリで避ける「ジャスト回避」において、単に無敵時間を付与するだけでは反撃のテンポが悪く、爽快感に欠ける問題がありました。
【解決】 回避成功時に、攻撃してきた敵の位置と向き（ForwardVector）を取得し、内積・外積等のベクトル演算を用いて「敵の背後（死角）の座標」を動的に算出するロジックをC++で実装しました。
【結果】 回避と同時に敵の背後へ自動で踏み込み、シームレスに反撃へ移行できるシステムが完成。コンセプトである「最大のピンチを最高のリターンにしろ！」を体現する、スピーディな切り返しアクションを実現しました。

### 2. 先行入力バッファとGASによる「手触り」の向上と安全なステート管理

【課題】 ハイスピードな戦闘において、ボタン連打による意図しないモーションの暴発を防ぎつつ、滑らかなコンボ連携を実現する必要がありました。
【解決】 アニメーションの特定フレームでのみ次アクションの予約を受け付ける「先行入力バッファ」を実装。さらにステート遷移のコアにGameplay Ability System (GAS) を導入し、各アクションを安全にカプセル化しました。
【結果】 プレイヤーの思考を阻害しない直感的でレスポンスの良い操作性を実現。また、フラグのスパゲッティ化を防ぎ、他メンバーが触っても意図しないバグが起きにくい、拡張性の高いアーキテクチャを構築しました。

## 🎮 操作方法 (Xboxコントローラー推奨)

本作はゲームパッドでのプレイを想定して直感的に操作できるよう調整されています。

| アクション | 入力ボタン (Xbox準拠) |
| :--- | :--- |
| **移動 / 選択** | 左スティック |
| **カメラ操作** | 右スティック |
| **ジャンプ / 決定** | A ボタン |
| **ダッシュ** | RB ボタン |
| **回避 / キャンセル** | B ボタン |
| **弱攻撃（連続入力でコンボ）** | X ボタン |
| **強攻撃（長押しで溜め）** | Y ボタン |
| **スキル** | RT ボタン |


## 🚀 インストールと実行方法

※ 本作の実行には Windows 環境が必要です。
1. GitHubの [Releases] ページから `Windows Build v1.0` (zipファイル) をダウンロードします。
2. ダウンロードした zip ファイルを任意のフォルダに展開します。
3. 展開したフォルダ内にある実行ファイル（.exe）を起動するとゲームが開始されます。
